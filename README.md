# 👗 Quintessa Modest Wear — E-Commerce Website

A fully responsive **multi-page e-commerce website** for a modest wear clothing brand, built with pure **HTML and CSS**. Designed to provide a smooth and elegant online shopping experience.

---

## 🌐 Pages Included

| Page | Description |
|------|-------------|
| 🏠 `index.html` | Homepage with hero section and featured products |
| 🛍️ `shop.html` | Full product catalog with browsing |
| 👗 `product.html` | Individual product detail page |
| 🛒 `cart.html` | Shopping cart with item management |
| 💳 `checkout.html` | Checkout form and order summary |
| 🔐 `auth.html` | Login and registration page |
| ℹ️ `about.html` | Brand story and information |
| 📞 `contact.html` | Contact form and details |

---

## ✨ Features

- 📱 **Responsive Design** — Works on desktop and mobile
- 🎨 **Clean UI** — Elegant and minimal styling
- 🛒 **Shopping Flow** — Browse → Product → Cart → Checkout
- 🔐 **Auth Pages** — Login and registration UI
- 🖼️ **Asset Structure** — Organized image folders per section

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3
- **Design:** Custom CSS styling
- **Structure:** Multi-page website architecture
